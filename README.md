💰 Finance Tracker

A simple and lightweight Finance Tracker Web App that helps users manage their income and expenses.
It allows you to add transactions, track your balance, and store data using browser local storage.

🚀 Features
➕ Add income and expense transactions
📊 Real-time balance calculation
🗂 Categorize transactions (Salary, Food, Shopping, etc.)
🗑 Delete transactions
💾 Data persistence using LocalStorage
🎨 Simple and clean UI

🛠 Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage API

📂 Project Structure
finance-tracker/
│── index.html   # Main file (HTML + CSS + JS)

⚙️ How It Works
User enters:
Amount
Type (Income / Expense)
Category

On submission:
Transaction is added to UI
Balance is updated
Data is stored in localStorage

On page reload:
Data is fetched from localStorage
Balance and transaction history are restored

▶️ How to Run
Clone the repository:
git clone https://github.com/ZaidKhan26/finance-tracker.git
Open the folder:
cd finance-tracker
Open index.html in your browser
